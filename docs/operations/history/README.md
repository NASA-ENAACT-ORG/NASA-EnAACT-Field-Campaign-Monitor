# Retired / History Docs

This folder holds Markdown docs that are preserved for historical context but
are no longer good entry points for current work.

## Structure

- `architecture/`
  - obsolete scheduler-era audits, verification notes, and older reorg plans
- `operations/`
  - outdated operating guides that no longer match the active runtime model

## Important Note

These files may still contain useful reasoning or old implementation details,
but many of them describe workflows that have since been replaced or retired.

For current docs, start in:

- `docs/operations/context/CURRENT_STATE.md`
- `docs/architecture/`
- `docs/operations/`
