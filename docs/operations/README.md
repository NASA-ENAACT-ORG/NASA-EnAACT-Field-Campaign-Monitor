# Operations Docs

This folder is organized by current use:

- `context/`
  - current project direction, cleanup priorities, and condensed history
- `guides/`
  - operational how-to documents that still describe supported workflows

## Historical Docs

Obsolete or heavily outdated operations docs have been moved to:

- `docs/retired/history/operations/`

Those files are preserved for reference, but they should not be followed as the
current operating model.
